Chatbot for Financial Analysis
------------------------------
This is a simple chatbot prototype that responds to predefined financial queries.

Queries Supported:
- "What is the total revenue?"
- "How has net income changed over the last year?"
- "What is the profit margin?"

Usage:
1. Run `python chatbot.py`
2. Open a browser and test queries like:
   http://127.0.0.1:5000/chatbot?query=What%20is%20the%20total%20revenue?
3. The chatbot returns predefined responses.

Limitations:
- Only supports predefined queries.
- No real-time financial data integration.
